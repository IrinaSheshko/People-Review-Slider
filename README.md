# 🎡 People Review Slider

Интерактивный компонент отзывов, созданный на **React**. Проект демонстрирует работу с динамическим изменением состояния и циклической навигацией по массиву данных.

## 🚀 [Live Demo](https://IrinaSheshko.github.io/People-Review-Slider/)

---

### 🌟 Основные функции:
* **Циклическая прокрутка:** Слайдер автоматически переходит к первому элементу после последнего (и наоборот).
* **Кнопка "Surprise Me":** Выбирает случайный отзыв, гарантируя, что он не повторится дважды подряд.
* **Анимация переходов:** Плавное появление контента при смене карточки (реализовано через React `key`).
* **Адаптивность:** Полная поддержка мобильных устройств.

---

### 🛠 Стек технологий:
* **React** (Hooks: `useState`, `useEffect`)
* **React Icons** (для стильных элементов управления)
* **CSS3** (Keyframes для анимаций, Flexbox для центрирования)
* **GitHub Pages** (деплой)
